# Hockey Rink Images

## Required Images

Place your hockey images in this folder with the following names:

### 1. Hero Section Image

- **Filename**: `hockey-player-hero.jpg`
- **Recommended size**: 1200x800px
- **Description**: Action shot of hockey players on the ice (skating, shooting, or game action)
- **Usage**: Main hero section on homepage

### 2. Team Photo

- **Filename**: `hockey-team.jpg`
- **Recommended size**: 1200x800px
- **Description**: Team photo or group of players celebrating/together
- **Usage**: "Join a Community" section on homepage

## Image Sources

You can find free hockey images from:

- [Unsplash](https://unsplash.com/s/photos/hockey)
- [Pexels](https://pexels.com/search/hockey/)
- [Pixabay](https://pixabay.com/images/search/hockey/)

## Format Recommendations

- **Format**: JPG or WebP
- **Quality**: High resolution (at least 1200px wide)
- **Orientation**: Landscape (horizontal)
- **Aspect Ratio**: 3:2 or 16:9

## Notes

- Images will automatically fall back to placeholder icons if not found
- Optimize images before uploading (use tools like TinyPNG or ImageOptim)
- Consider using WebP format for better performance
